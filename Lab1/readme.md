#Laborator 1 - Introducere în limbajul de programare C
#Descriere Laborator 1
1. Lucrarea urmărește familiarizarea studentului cu limbajul C și utilizarea operatorilor aritmetici și logici.

2. Se studiază funcțiile de intrare/ieșire (scanf, printf) și funcțiile matematice precum sqrt, log, pow.

3. Programul implementează un mini-calculator pentru două variabile și verifică condiții logice folosind operatori precum && și ||.

4. Se realizează calcule suplimentare pentru aria și volumul unui cilindru, precum și evaluarea unei expresii matematice complexe.
